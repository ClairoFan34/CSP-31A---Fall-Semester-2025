1\. A **constant** is a memory location whose value cannot change during the execution of a program

2\. **Integer** is a data type that only holds numbers with no fractional component

3\. **Float** is a data type that only holds numbers with fractional components

4\. **Modulus** is an arithmetic operator that gives remainder of a division problem

5\. cout << is an example of the **output** fundamental instruction

6\. **Boolean** data types only have two values: true and false

7\. One byte consist of **8** bits

8\. // or /\* in C++ indicates the start of a **comment**

9\. A **variable** is a memory location whose value can change during the execution of the program

10\. A **string** can hold a sequence of character such as a name
